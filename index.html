Android bug list

By kobeli

Published 2018-04-09

NO 1:

今天用一台Android4.4.4的手机跑neuron的Android版本，老是报这个错：

java.lang.RuntimeException: 
Unable to get provider com.google.firebase.provider.FirebaseInitProvider: 
java.lang.ClassNotFoundException: Didn't find class  
"com.google.firebase.provider.FirebaseInitProvider" on path:

解决方法：
可能是添加MultiDex分包，但未初始化的原因，在Application中重写attachBaseContext函数，对MultiDex初始化即可。

1.首先修改Gradle配置文件，启用MultiDex并包含MultiDex支持：

defaultConfig {
    multiDexEnabled true
 }

2.设置Application

public class App extends Application {
    @Override
    protected void attachBaseContext(Context base) {
         super.attachBaseContext(base);
      MultiDex.install(base);
    }
}
